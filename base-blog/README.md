## 已经完成功能

-  登录  
-  注册  
-  文章列表
-  标签  
-  关于  
-  点赞与评论
-  文章详情（支持代码语法高亮）
-  文章详情目录

## 前端用到的框架与工具 
- vue
- typescript
- element-plus
- vue-router
- vite
- vuex
- axios
- highlight.js
- marked

## 获取项目代码

```
git clone git@github.com:Watcats/my-blog.git
```

## Build Setup 

```
# install dependencies
npm install
```

```
# Compiles and hot-reloads for development
npm run dev
```

```
# Compiles and minifies for production
npm run build
```
